Dial-Up to Connect to the Campus Network

Astrid Stark Lv. ∞

一、设置新拨号连接

打开“网络和 Internet”

点击“拨号”

点击“设置新连接”

点击“连接到 Internet”

点击“设置新连接”

点击“否，创建新连接(C)”

点击“宽带(PPPoE)(R)”

输入用户名和密码，进行连接测试

连接名称只能包含英文字母、-_或者数字，否则代理设置无法识别

二、设置启动自动连接

使用 Win+R 并输入 taskschd.msc 打开任务计划程序

点击右侧的“创建任务”

输入名称并点击“使用最高权限运行”

新建触发器，选择“登录时”作为触发条件

新建操作，选择“启动程序”，程序或脚本填入 rasdial ，添加拨号连接参数。

参数格式：连接名 用户名 密码

点击“确定”，保存任务

三、疑难解答

如果出现在连接时蓝屏，Stop code 为 BUGCODE_NDIS_DRIVER 的情况，卸载 Npcap 并重试。
